Releases

There have been no official releases of Apache Wave currently.

In the meantime, please use the code from source, or refer to our last release candidates (for pre-builds) by checking the wave-dev mailing list archives. The latest release candidate of Wave is 0.4-rc10 (posted on 10 October 2014).

Once compiled, execute the run-server.sh or run-server.bat to start the Wave server.

When downloading from a mirror, please be sure to verify the checksums and signature (see the MD5 and PGP links above). The KEYS file contains the public key(s) used for signing releases.
