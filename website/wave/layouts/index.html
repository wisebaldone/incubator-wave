Apache Wave is a distributed, near-real-time, rich collaboration platform that allows users to work together in new and exciting ways. Wave allows for flexible modes of communication, blending chat, email and collaborative document editing in to one seamless environment; providing a lively and responsive environment that promotes more fluid and dynamic collaboration between users.

Wave is built upon Operation Transforms which power real time systems like Google Docs. The projects aim is to provide this technology to many users through a backend server and a REST Api; much like the Realtime Api offered by Google but for "Host Your Own" solutions.
